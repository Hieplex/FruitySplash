# Fruity Splash Build And Run

## Install

```powershell
npm install
```

## Start Metro

```powershell
npm run start
```

## Run Android

```powershell
npm run android
```

## Run iOS

```powershell
npm run ios
```

## Typecheck

```powershell
npm run typecheck
```

## Tests

```powershell
npm run test
npm run verify:levels
```
