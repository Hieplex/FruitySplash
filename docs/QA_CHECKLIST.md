# Fruity Splash QA Checklist

- App launches from home screen without crash
- Level map loads and level 1 is unlocked
- Entering a level shows board, timer, and score HUD
- Invalid swaps revert cleanly
- Valid matches clear and refill correctly
- Cascades resolve without hanging
- Win flow awards stars and unlocks next level
- Loss flow returns cleanly and allows retry
- Saved stars persist after restart
- Portrait layout remains readable on phone
