# Fruity Splash Implementation Plan

> **For agentic workers:** REQUIRED SUB-SKILL: Use superpowers:subagent-driven-development (recommended) or superpowers:executing-plans to implement this plan task-by-task. Steps use checkbox (`- [ ]`) syntax for tracking.

**Goal:** Build a runnable, polished Expo mobile match-3 game called `Fruity Splash` with a fresh natural fruit theme, time-limit levels, star progression, and 120 levels in portrait one-hand mobile format.

**Architecture:** The app shell, puzzle engine, level data, asset catalog, and progression save layer stay separate. Gameplay rules live in a deterministic engine module, while React Native screens adapt state and animation around that engine. Generated assets and UI motion are integrated through a manifest layer so the game can be expanded later with blockers, ads, and monetization without rewriting the core.

**Tech Stack:** Expo, React Native, TypeScript, Expo Router, React Native Reanimated, AsyncStorage, local JSON/TS level data, image assets generated by GPT images.
